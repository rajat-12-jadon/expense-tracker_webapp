# 💰 Expense Tracker Web App

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript** with **AI-assisted development using Antigravity Studio**.

This application allows users to track their income and expenses, categorize transactions, visualize spending patterns, and maintain persistent data using **Local Storage**.

---

## 🚀 Features

- Add income and expense transactions
- Categorize transactions (Food, Transport, Shopping, Bills, Salary, Other)
- Real-time **balance, income, and expense calculation**
- **Expense visualization using Chart.js**
- Persistent data storage using **Local Storage**
- **Dark mode / Light mode toggle**
- Animated UI interactions
- Fully **responsive design for desktop, tablet, and mobile**

---

## 🛠️ Technologies Used

- **HTML5** – Structure
- **CSS3** – Styling and responsive design
- **JavaScript (ES6)** – Application logic
- **Chart.js** – Data visualization
- **Font Awesome** – Icons
- **Google Fonts (Inter)** – Typography
- **Local Storage API** – Data persistence

---

## 📊 How It Works

1. Users add a transaction with:
   - description
   - amount (+ income / - expense)
   - category

2. Transactions are stored in **Local Storage**.

3. The app automatically updates:
   - Total balance
   - Total income
   - Total expenses

4. Expenses are grouped by category and visualized using a **doughnut chart**.

---

## 📱 Responsive Design

The UI adapts to different screen sizes using **CSS media queries**:

- Desktop
- Laptop
- Tablet
- Mobile devices

---

## 🌙 Dark Mode

Users can toggle between **light mode and dark mode**, and the preference is saved using Local Storage.
